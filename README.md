# Smart Parking Management System 🚗

A web-based parking management system developed using PHP and MySQL to automate vehicle parking operations in residential societies or commercial buildings.

## Features

* Admin and Security login panels
* Resident vehicle entry/exit
* Guest vehicle entry/exit
* Parking slot allocation and status tracking
* Resident and guest parking logs
* Vehicle and guest search
* Daily and parking reports
* Security staff management

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP
